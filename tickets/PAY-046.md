# PAY-046: ComputePayslipAction - Absent and Statutory Deductions (Steps 3-6)

## Description

Implement Steps 3 through 6 of the 10-step payroll computation pipeline: absent deduction, SSS contribution, PhilHealth contribution, and Pag-IBIG contribution. All four steps use Step 1's `gross_pay` (basic salary) as the computation basis -- not the adjusted gross after bonuses. These steps are skipped entirely for employees with `has_external_salary = true`.

## Technical Implementation

### Action

**`app/Payroll/Actions/ComputePayslipAction.php`** (Steps 3-6 segment)

All steps are skipped if `$employee->has_external_salary === true`.

---

**Step 3 -- Absent Deduction:**

```php
if ($compensation->salary_basis === 'monthly') {
    $dailyRateForDeduction = $compensation->monthly_salary * $snapshotRate / 22;
} else {
    $dailyRateForDeduction = $compensation->daily_rate * $snapshotRate;
}

$absentDeduction = $dailyRateForDeduction * $payslip->absent_days;
```

Record line item: `{ code: 'ABSENT', type: 'deduction', amount: $absentDeduction }`

The divisor of 22 represents standard working days per month for monthly-basis employees.

---

**Step 4 -- SSS Contribution:**

SSS uses a bracket-based lookup table stored in `config/payroll.php` under `sss_contribution_table`. The table maps salary credit ranges to employee share amounts.

```php
$sssBasis = $grossPay; // Step 1 gross, not adjusted
$sssEmployeeShare = SssLookup::getEmployeeShare($sssBasis);
```

The lookup scans the SSS contribution table for the bracket where `$sssBasis` falls between `range_from` and `range_to`, then returns `employee_share`.

Record line item: `{ code: 'SSS', type: 'deduction', amount: $sssEmployeeShare }`

**SSS Table (stored in `config/payroll.php`):**

```php
'sss_contribution_table' => [
    ['range_from' => 0,      'range_to' => 4249.99,  'employee_share' => 180.00],
    ['range_from' => 4250,   'range_to' => 4749.99,  'employee_share' => 202.50],
    // ... full SSS 2024 table (approximately 30 brackets)
    ['range_from' => 29750,  'range_to' => 99999999, 'employee_share' => 1350.00],
],
```

---

**Step 5 -- PhilHealth Contribution:**

```php
$philhealthBasis = max(10000, min($grossPay, 100000)); // Floor 10k, ceiling 100k
$philhealthDeduction = $philhealthBasis * 0.025; // 2.5% employee share
```

Record line item: `{ code: 'PHILHEALTH', type: 'deduction', amount: $philhealthDeduction }`

The total PhilHealth rate is 5% (shared equally between employer and employee at 2.5% each). Only the employee share is deducted.

---

**Step 6 -- Pag-IBIG Contribution:**

```php
$pagibigDeduction = 100.00; // Fixed PHP 100 employee share
```

Record line item: `{ code: 'PAGIBIG', type: 'deduction', amount: $pagibigDeduction }`

---

**Update payslip with individual deduction amounts:**

```php
$payslip->update([
    'absent_deduction'      => $absentDeduction,
    'sss_contribution'      => $sssEmployeeShare,
    'philhealth_contribution' => $philhealthDeduction,
    'pagibig_contribution'  => $pagibigDeduction,
]);
```

### Helper Class

**`app/Payroll/Support/SssLookup.php`**

```php
class SssLookup
{
    public static function getEmployeeShare(float $monthlySalaryCredit): float
    {
        $table = config('payroll.sss_contribution_table');

        foreach ($table as $bracket) {
            if ($monthlySalaryCredit >= $bracket['range_from']
                && $monthlySalaryCredit <= $bracket['range_to']) {
                return $bracket['employee_share'];
            }
        }

        // Fallback to max bracket
        return $table[count($table) - 1]['employee_share'];
    }
}
```

### Edge Cases

- Zero absent days: absent deduction = 0, line item still created with amount 0 (or skipped -- prefer skipping zero-amount line items for cleanliness).
- Gross pay below SSS minimum bracket: returns the minimum employee share.
- Gross pay above SSS maximum bracket: returns the maximum employee share.
- Gross pay below PhilHealth floor (< 10,000): basis is floored to 10,000.
- Gross pay above PhilHealth ceiling (> 100,000): basis is capped at 100,000.

## Acceptance Criteria

- [ ] Absent deduction uses `monthly_salary / 22` as daily rate for monthly-basis employees.
- [ ] Absent deduction uses `daily_rate` directly for daily-basis employees.
- [ ] Absent deduction = `daily_rate_for_deduction * absent_days`.
- [ ] SSS bracket lookup returns the correct employee share for the given gross pay.
- [ ] SSS uses Step 1 `gross_pay` as basis, not adjusted gross.
- [ ] PhilHealth applies 2.5% employee share with a floor of 10,000 and ceiling of 100,000.
- [ ] PhilHealth uses Step 1 `gross_pay` as basis.
- [ ] Pag-IBIG deduction is always fixed at PHP 100.00.
- [ ] All four deductions are recorded as `deduction`-type line items.
- [ ] Individual deduction amounts are stored on the payslip record.
- [ ] Steps 3-6 are skipped entirely for employees with `has_external_salary = true`.
- [ ] SSS contribution table is stored in `config/payroll.php` and is easily updatable.
