# PAY-029: Employee External Salary and Payroll Flags

## Description

As an owner, I want to manage payroll-specific flags on employee profiles (has_external_salary, has_mp2, has_sss_wisp, hmo_amount) so that payroll computation handles each employee correctly. The `has_external_salary` flag is owner-only and skips all deductions/tax when enabled.

## Technical Implementation

### Action
- **`app/Employee/Actions/UpdateEmployeePayrollFlagsAction.php`** (lorisleiva/laravel-actions)
  - Accepts: `Employee $employee`, `array $flags`, `User $updatedBy`
  - Updates: `has_external_salary`, `has_mp2`, `has_sss_wisp`, `hmo_amount`
  - Authorization: `has_external_salary` requires owner role; `has_mp2`, `has_sss_wisp`, `hmo_amount` allow payroll_officer or owner

### Policy
- **`EmployeePolicy@updatePayrollFlags`** ‚Äî owner and payroll_officer
- **`EmployeePolicy@updateExternalSalary`** ‚Äî owner only

### Form Request
- **`UpdatePayrollFlagsRequest`**
  - `has_external_salary`: boolean (only if user is owner)
  - `has_mp2`: boolean
  - `has_sss_wisp`: boolean
  - `hmo_amount`: numeric, min:0

### Controller
- **`EmployeeController@updateFlags`** ‚Äî `PATCH /employees/{employee}/flags`

### Frontend
- On employee profile page, a "Payroll Settings" section with:
  - Toggle: `has_external_salary` (visible only to owner, with confirmation dialog)
  - Toggle: `has_mp2`
  - Toggle: `has_sss_wisp`
  - Number input: `hmo_amount` (‚Ç±)
- Confirmation dialog for external salary: "This will skip all deductions and tax for this employee. Continue?"

### Audit
- All flag changes audited via owen-it/laravel-auditing on Employee model
- Audit captures old/new values for each flag

### Tests
- Owner can toggle all flags
- Payroll officer can toggle has_mp2, has_sss_wisp, hmo_amount but NOT has_external_salary
- Finance and employee roles cannot update flags
- Audit trail captures all changes

## Acceptance Criteria

- [ ] Only owner can toggle `has_external_salary`
- [ ] Payroll officer can toggle `has_mp2`, `has_sss_wisp`, and update `hmo_amount`
- [ ] Toggling `has_external_salary` shows confirmation dialog
- [ ] All flag changes appear in audit trail with old/new values
- [ ] Finance and employee roles get 403 on flag update
- [ ] `hmo_amount` validates as numeric >= 0
