# [GW] Payroll - Form Components

## Description

- Create library of reusable form fields.
- Wrap labels, validate errors, and descriptions.
- Integrate smoothly with Inertia form handles.
- Display server-side errors perfectly inline.
- Support unified usage across all forms.

## Technical Implementation

### Edge Cases & Notes

- Components gracefully ignore missing error values.
- All inputs provide strict accessibility labels.
- Number inputs firmly reject non-numeric pastes.
- Date picker stores values standard ISO.
- Section components support visual nesting hierarchies.

## Acceptance Criteria

### Client-side

#### Layout

- Fields render label, input, optional description.

#### Behavior

- Text input supports text, email, password.
- Number input enforces precise numeric values.
- Select renders clear dropdown option list.
- Date picker selects standard calendar dates.
- Toggle renders boolean switch configuration control.
- Textarea configures adjustable vertical multiline heights.
- Sections visually separate grouped related fields.
- Forms render bottom submit cancel buttons.

#### Validation

- Server errors display below corresponding fields.
- Errors render prominent red warning text.
- Submit button shows dynamic loading indicator.
- Processing forms inherently disable submit button.
- Field components link to Inertia flawlessly.
