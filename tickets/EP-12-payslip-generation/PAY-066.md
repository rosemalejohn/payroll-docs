# [GW] Payroll - Payslip Detail View with Deduction and Bonus Breakdown

## Description

Build the employee-facing payslip detail page that displays the full breakdown of a finalized payslip. The view includes an earnings section (basic salary plus itemized bonuses), a deductions section (itemized with computation basis descriptions), and a summary section. Each deduction line shows a brief description of the computation basis. Voluntary deductions are labeled, and overridden deductions are marked with an asterisk. Foreign currency employees see dual amounts with exchange rates.

## Technical Implementation

### Routes & Controllers

- `GET /my-payslips/{payslip}` — payslip detail page with full breakdown. Eager loads employee, payroll run, payroll period, line items, and currency snapshot.

### API Resources

- Payslip Detail resource: includes employee info, period, finalized date, pay basis, earnings (basic salary plus itemized bonuses), deductions (itemized with labels, basis descriptions, amounts, voluntary flags, and override indicators), summary (gross, bonuses, deductions, net), and currency conversion data for foreign salary employees.

### Policies & Authorization

- Employee can only view their own payslip.
- Payslip must belong to a finalized payroll run.
- Non-finalized payslips return 403.

### Tests

- Assert employee can view their own finalized payslip with full detail.
- Assert all earnings line items are displayed in correct order.
- Assert all deductions are displayed with correct labels, amounts, and basis descriptions.
- Assert voluntary deductions are labeled as "Voluntary".
- Assert overridden deductions are marked with an asterisk and include the override note.
- Assert foreign currency payslip shows original amount, PHP equivalent, and exchange rate.
- Assert summary totals are mathematically correct.
- Assert employee cannot view another employee's payslip (403).
- Assert employee cannot view a non-finalized payslip (403).

### Edge Cases & Notes

- Deduction order is consistent: SSS, PhilHealth, Pag-IBIG, HMO, BIR Tax, then voluntary deductions.
- Each deduction has a human-readable basis description (e.g., "Bracket-based (employee share)" for SSS, "2.5% of basic salary" for PhilHealth, "Fixed contribution" for Pag-IBIG).
- If an employee has no bonuses, the bonuses subsection displays an empty state.
- All monetary values formatted with thousands separators and 2 decimal places.

## Acceptance Criteria

### Client-side

#### Layout

- The payslip detail page has sections in order: Header (employee info, period), Earnings, Deductions, Summary, and Currency Info (if foreign currency)

#### Behavior

- All earnings line items (basic salary plus bonuses) are displayed with labels and amounts
- All deductions are displayed with labels, computation basis descriptions, and amounts
- Voluntary deductions show a "Voluntary" badge
- Overridden deductions show an asterisk with a tooltip or note explaining the adjustment
- Foreign currency payslips show the original currency amount, PHP equivalent, and exchange rate
- The summary section shows gross pay, total bonuses, total deductions, and net pay (highlighted)
- A Download PDF button is available at the top of the page

#### Validation

- Accessing a non-finalized payslip returns an error
- Accessing another employee's payslip returns an error

---

### Server-side

#### Side-effects

- Employees can only view their own finalized payslips
- Non-finalized payslips are denied access
- Deduction basis descriptions are included for each deduction type
- All monetary values use consistent formatting
