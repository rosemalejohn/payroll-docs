# PAY-065: Employee Payslip List Page

## Description

Implement the employee-facing payslip list page where authenticated employees can view their own finalized payslips. The list displays key payslip summary information (period, gross pay, total deductions, net pay) with a download button for each entry. Only finalized payslips are visible; draft, submitted, and under_review payslips are completely hidden from employees. Direct URL access to non-finalized payslips returns 403.

## Technical Implementation

### Routes & Controllers
- **`GET /my-payslips`** — Route name: `employee.payslips.index`
- **`app/Http/Controllers/Web/EmployeePayslipController@index`**
  - Middleware: `auth`, `role:employee`
  - Queries payslips belonging to the authenticated employee where the associated payroll run is in the `finalized` state
  - Sorts results by payroll period descending (newest first)
  - Paginates results (default 15 per page)
  - Returns Inertia response with payslip summary data

### Queries
- **`app/Payroll/Queries/EmployeePayslipsQuery.php`** — Dedicated query class:
  - Filters payslips by `employee_id` matching the authenticated user's linked employee record
  - Joins payroll run to filter only `finalized` state
  - Eager loads payroll period for display
  - Orders by `payroll_periods.start_date` descending
  - Returns paginated results

### API Resources
- **`app/Http/Resources/Payroll/EmployeePayslipListResource.php`** — Shapes the response:
  - `id`: payslip ULID
  - `period`: formatted payroll period (e.g., "March 1-15, 2026")
  - `gross_pay`: formatted decimal
  - `total_deductions`: formatted decimal
  - `net_pay`: formatted decimal
  - `has_pdf`: boolean indicating if `pdf_path` is set
  - `finalized_at`: formatted date

### Policies & Authorization
- **`app/Payroll/Policies/PayslipPolicy.php`** — Add/update `viewAny()` method:
  - Employees can only list their own payslips
  - The query itself enforces the ownership filter
- Ensure existing `view()` and `download()` policies reject access to non-finalized payslips for employees

### Frontend Components
- **`resources/js/pages/employee/payslips/index.tsx`** — Inertia page component:
  - Page title: "My Payslips"
  - Uses authenticated layout
  - Renders a data table with columns: Period, Gross Pay, Total Deductions, Net Pay, Actions
  - Actions column: "Download PDF" button (links to `payslips.download` route)
  - Download button disabled with tooltip if PDF is not yet generated
  - Empty state message if no finalized payslips exist
  - Pagination controls at the bottom
- **`resources/js/components/tables/payslip-list-table.tsx`** — Reusable table component for the payslip list

### Tests
- **`tests/Feature/Payroll/EmployeePayslipListTest.php`**
  - Employee sees only their own finalized payslips
  - Employee does not see payslips from draft, submitted, or under_review runs
  - Employee does not see other employees' payslips
  - Payslips are sorted by period descending (newest first)
  - Pagination works correctly
  - Download button is present for payslips with generated PDFs
  - Direct URL to a non-finalized payslip returns 403
  - Direct URL to another employee's payslip returns 403
  - Non-employee roles (owner, payroll_officer, finance) are redirected or see their own role-appropriate view
  - Unauthenticated users are redirected to login

### Edge Cases & Notes
- The employee's `user_id` must be correctly linked to their `employee_id` in the payroll system (via HRIS sync)
- If an employee has no finalized payslips, display a friendly empty state
- PDF download availability depends on the async PDF generation job (PAY-063) completing; show appropriate state if PDF is still pending
- Monetary values should use consistent formatting with thousands separators and 2 decimal places

## Acceptance Criteria

- [ ] Only finalized payslips are shown to employees
- [ ] Employees can only see their own payslips (not other employees')
- [ ] Payslips are sorted by payroll period in descending order (newest first)
- [ ] Each payslip row shows: period, gross pay, total deductions, net pay
- [ ] Download PDF button is available and functional for each payslip
- [ ] Direct URL access to a non-finalized payslip returns 403
- [ ] Direct URL access to another employee's payslip returns 403
- [ ] Pagination is functional
- [ ] Empty state is shown when no finalized payslips exist
