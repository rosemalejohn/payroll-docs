# [GW] Payroll - ComputePayslipAction - Net Pay and External Salary Path (Step 10)

## Description

Implement Step 10 of the 10-step payroll computation pipeline: final net pay calculation and the external salary shortcut path. This step sums all deductions, computes net pay (floored at zero), sets a warning flag if the floor is triggered, and handles the special case where employees with external salaries skip all deduction steps (3-9).

## Technical Implementation

### Action

**`app/Payroll/Actions/ComputePayslipAction.php`** (Step 10 segment)

---

**Standard path (has_external_salary = false):**

```php
$totalDeductions = $absentDeduction
    + $sssEmployeeShare
    + $philhealthDeduction
    + $pagibigDeduction
    + $hmoAmount
    + $mp2Amount
    + $wispAmount
    + $cashAdvanceTotal
    + $monthlyBirTax;

$netPay = $grossPay + $totalBonuses - $totalDeductions;

// Floor at zero
$netPayFloored = false;
if ($netPay < 0) {
    $netPay = 0;
    $netPayFloored = true;
}

$payslip->update([
    'total_deductions'  => $totalDeductions,
    'net_pay'           => $netPay,
    'net_pay_floored'   => $netPayFloored,
]);
```

---

**External salary path (has_external_salary = true):**

When `$employee->has_external_salary === true`, Steps 3 through 9 are skipped entirely. The employee's gross pay and bonuses are still computed (Steps 1-2), but no deductions are applied.

```php
if ($employee->has_external_salary) {
    $totalDeductions = 0;
    $netPay = $grossPay + $totalBonuses;

    $payslip->update([
        'total_deductions'       => 0,
        'net_pay'                => $netPay,
        'net_pay_floored'        => false,
        'absent_deduction'       => 0,
        'sss_contribution'       => 0,
        'philhealth_contribution' => 0,
        'pagibig_contribution'   => 0,
        'hmo_deduction'          => 0,
        'mp2_deduction'          => 0,
        'sss_wisp_deduction'     => 0,
        'cash_advance_deduction' => 0,
        'bir_tax'                => 0,
        'is_external_salary'     => true,
    ]);
}
```

No deduction line items are created for external salary employees.

---

**Final payslip field update (all employees):**

Ensure all computed fields are populated on the payslip record:

```php
$payslip->update([
    'gross_pay'         => $grossPay,
    'total_bonuses'     => $totalBonuses,
    'total_deductions'  => $totalDeductions,
    'net_pay'           => $netPay,
    'net_pay_floored'   => $netPayFloored,
    'computed_at'       => now(),
]);
```

### Warning Flag

When `net_pay_floored = true`, the UI should surface a visual warning on the payslip row (e.g., orange badge or icon) indicating that the employee's deductions exceeded their earnings. This flag helps payroll officers and finance reviewers identify anomalies before finalization.

### Control Flow Summary

```
ComputePayslipAction($payslip):
  Step 1:  Gross Pay (all employees)
  Step 2:  Bonuses (all employees)

  if has_external_salary:
    -> net_pay = gross_pay + total_bonuses
    -> total_deductions = 0
    -> DONE

  Step 3:  Absent Deduction
  Step 4:  SSS
  Step 5:  PhilHealth
  Step 6:  Pag-IBIG
  Step 7:  HMO
  Step 8:  Voluntary (MP2, WISP, Cash Advance)
  Step 9:  BIR Tax

  Step 10: net_pay = gross_pay + total_bonuses - total_deductions
           net_pay = max(0, net_pay)
```

### Edge Cases

- Deductions exceed gross + bonuses: net pay floored to 0, `net_pay_floored = true`.
- External salary with no bonuses: net pay = gross pay.
- External salary with bonuses: net pay = gross pay + total bonuses.
- Zero gross pay (edge case): net pay = total bonuses - total deductions (floored at 0).

## Acceptance Criteria

- [ ] Net pay is computed as `gross_pay + total_bonuses - total_deductions`.
- [ ] Net pay never goes below zero (floored at 0).
- [ ] When net pay is floored to zero, `net_pay_floored = true` is set on the payslip.
- [ ] A visual warning is surfaced in the UI for floored payslips.
- [ ] External salary employees (`has_external_salary = true`) skip Steps 3-9 entirely.
- [ ] External salary employees have `total_deductions = 0` and all individual deduction fields set to 0.
- [ ] External salary employees have `net_pay = gross_pay + total_bonuses`.
- [ ] No deduction line items are created for external salary employees.
- [ ] All computed fields are populated on the payslip: `gross_pay`, `total_bonuses`, `total_deductions`, `net_pay`, `computed_at`.
- [ ] The `is_external_salary` flag is recorded on the payslip for external salary employees.
