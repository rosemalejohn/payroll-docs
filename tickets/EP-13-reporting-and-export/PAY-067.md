# [GW] Payroll - Payroll Summary Report Page

## Description

Build a payroll summary report page accessible to Owner and Finance roles for finalized payroll runs. The report aggregates all payslip data into a company-level summary: total gross pay, total deductions broken down by type, total net pay, and employer contributions. The report supports filtering by department and pay basis, and can be exported as CSV or PDF.

## Technical Implementation

### Queries

- Payroll summary query: accepts a payroll run ID with optional department and pay basis filters. Aggregates: total gross pay, per-deduction-type totals (SSS, PhilHealth, Pag-IBIG, HMO, MP2, WISP, cash advance, BIR), total deductions, total net pay, employer contributions, and total employer cost.

### Routes & Controllers

- GET /reports/payroll-summary/{payrollRun} — summary report with optional department and pay basis filters.

### API Resources

- Payroll Summary resource: includes run details, employee count, all aggregated totals, employer contributions, total employer cost, and available filter options.

### Policies & Authorization

- Only Owner and Finance roles can view reports.
- Payroll run must be in finalized state.

### Tests

- Assert Owner can view the summary report for a finalized run.
- Assert Finance can view the summary report for a finalized run.
- Assert Payroll Officer and Employee roles receive 403.
- Assert non-finalized run returns 403.
- Assert total gross pay matches the sum of all payslip gross pay values.
- Assert total deductions match the sum of individual deduction types.
- Assert employer contributions are calculated correctly.
- Assert department and pay basis filters narrow results correctly.

### Edge Cases & Notes

- External salary employees are excluded from deduction totals.
- Use DECIMAL math for all aggregated amounts.
- The department list is dynamically pulled from employees in the current run.

## Acceptance Criteria

### Client-side

#### Layout

- The Payroll Summary Report is accessible from the Reports section for Owner and Finance roles

#### Behavior

- The report displays summary cards for total gross pay, total deductions, total net pay, and total employer cost
- A deductions breakdown table shows each deduction type with employee share totals and employer share totals (where applicable)
- An employer contributions section shows SSS, PhilHealth, and Pag-IBIG employer shares
- Filters for department and pay basis allow narrowing the report scope
- Export CSV and Export PDF buttons are available

---

### Server-side

#### Side-effects

- The report is available only for finalized payroll runs
- Only Owner and Finance roles can access the report
- All monetary totals match the sum of individual payslip values exactly
- Department and pay basis filters correctly narrow the aggregated results
- Totals use database-level aggregation for accuracy
