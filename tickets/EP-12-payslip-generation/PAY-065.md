# [GW] Payroll - Employee Payslip List Page

## Description

Build the employee-facing payslip list page where authenticated employees can view their own finalized payslips. The list displays key payslip summary information with a download button for each entry. Only finalized payslips are visible; draft, submitted, and under review payslips are completely hidden from employees.

## Technical Implementation

### Queries

- Employee payslips query: filters payslips by the authenticated employee, joins payroll run to show only finalized state, eager loads payroll period, orders by period start date descending, and returns paginated results.

### Routes & Controllers

- GET /my-payslips — employee payslip list, paginated (15 per page).

### API Resources

- Employee Payslip List resource: includes payslip ID, formatted payroll period, gross pay, total deductions, net pay, PDF availability flag, and finalized date.

### Policies & Authorization

- Employees can only list their own payslips.
- The query enforces the ownership filter.
- Non-finalized payslips are completely hidden from the Employee role.

### Tests

- Assert employee sees only their own finalized payslips.
- Assert employee does not see payslips from draft, submitted, or under review runs.
- Assert employee does not see other employees' payslips.
- Assert payslips are sorted by period descending (newest first).
- Assert pagination works correctly.
- Assert download button is present for payslips with generated PDFs.
- Assert direct URL access to a non-finalized payslip returns 403.
- Assert direct URL access to another employee's payslip returns 403.

### Edge Cases & Notes

- Show a friendly empty state if the employee has no finalized payslips.
- PDF availability depends on the async generation job completing.
- Format monetary values with thousands separators and 2 decimal places.

## Acceptance Criteria

### Client-side

#### Layout

- The "My Payslips" page is accessible from the sidebar for the Employee role

#### Behavior

- The payslip list displays: Period, Gross Pay, Total Deductions, Net Pay, and a Download PDF button
- Only finalized payslips are shown — draft, submitted, and under review payslips are hidden
- Payslips are sorted by payroll period in descending order (newest first)
- The list is paginated
- The Download PDF button is disabled with a tooltip if the PDF has not yet been generated
- A friendly empty state message is shown if no finalized payslips exist

---

### Server-side

#### Side-effects

- Employees can only see their own finalized payslips
- Non-finalized payslips are completely hidden from the Employee role
- Direct URL access to a non-finalized or another employee's payslip is denied
