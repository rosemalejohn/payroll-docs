# [GW] Payroll - Data Table Component

## Description

- Create reusable server-side data table reliably.
- Support pagination, sorting, and advanced filtering.
- Utilize URL query parameter state completely.
- Serve across all application list pages.
- Handle component empty states very gracefully.

## Technical Implementation

### Edge Cases & Notes

- All data fetching remains server-side strictly.
- Search input applies 300ms debounce effectively.
- Filter changes explicitly reset page 1.
- Table handles varied column counts effortlessly.
- URL parameters remain bookmarkable and encoded.

## Acceptance Criteria

### Client-side

#### Layout

- Table shows filter bar, header, rows, pagination.

#### Behavior

- Clickable headers sort columns comprehensively.
- Sort cycles ascending, descending, and default.
- Active sort column displays arrow indicator.
- Pagination shows numbers, previous, next buttons.
- Pagination summarizes active item query results.
- Page size selector allows 10 to 100.
- Search input filters actively during typing.
- Dropdown filters narrow specific result categories.
- Clear filters button resets all parameters.
- Changing filters resets to page 1.
- Changing pages preserves active existing filters.
- Pagination attributes remain in sharable URL.
- Filtering displays standard zero-results empty state.
- Empty data displays different initial state.
