# [GW] Payroll - Authorization & Roles

## Description

- Install Spatie permissions package easily.
- Implement four-role authorization control system.
- Add owner, payroll_officer, finance, employee.
- Create core role definitions and seeders.
- Provide role checking middleware accurately.
- Deny finalization to Payroll Officers unconditionally.
- Officers receive notification regarding finalized runs.
- Restrict employees to strictly finalized payslips.
- Hide non-finalized lists and endpoints.
- Employees receive notifications after valid finalization.

#### Layout

- Employees access My Payslips navigation easily.

#### Behavior

- Finalize button remains completely hidden.
- Officer views current pending run status.
- Officer receives reliable in-app notification.
